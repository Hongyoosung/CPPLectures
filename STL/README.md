# Standard Template Libarary

## 학습 목표

* STL에 대해서 이해하고 설명할 수 있다.
* STL의 컨테이너, 반복자, 알고리즘에 대해서 이해하고 설명할 수 있다.
* STL 컨테이너인 시퀀스 컨테이너, 연관 컨테이너, 컨테이너 어뎁터의 분류를 이해하고 설명할 수 있다.
* SLT의 대표적인 컨테이너인 vector, map, list, queue, stack을 이해하고 설명할 수 있다.
